셀렉터 다루기

  • 페이지가 렌더링된 후 CSS 변경하기
  • 준비 핸들러 사용하기
  • $, jQuery 출력하기
  • 엘리먼트에 접근해서 스타일 지정하기-“element”, “#id”, “.class
  • 한 번에 다양한 엘리먼트에 접근하여 개수와 텍스트 얻기-“selector1, selector2, selectorN”
  • 선택된 jQuery 확장 객체 집합에서 자손 엘리먼트에 스타일시트 적용하기-“ancestor descendant”
  • 인접한 자손 엘리먼트를 노드로 추가하기-“parent < child”
  • 모든 엘리먼트에 스타일을 적용하고 인접한 형제를 노드로 추가하기-“*”, “prev + next”
  • 조건에 만족하는 첫 번째와 마지막 엘리먼트에 스타일을 적용하기-:first, :last
  • 조건에 만족하는 짝수와 홀수에 위치한 엘리먼트에 스타일을 적용하기-:odd, :even
  • n번째로 일치하는 엘리먼트에 스타일 적용하기-:eq(n)
  • n번째 엘리먼트 이후 혹은 이전에 위치한 엘리먼트에 스타일 해제하기-:gt(n),:lt(n)
  • 첫 번째와 마지막 자식 엘리먼트에 스타일 적용하기 :first-child, :last-child
  • n번째 혹은 짝수 또는 홀수 자식 엘리먼트 검색하기-:nth-child(n), :nth-child(even), :nth-child(old)
  • 조건에 반대되는 엘리먼트 검색하기-:not()
  • 어트리뷰트를 조건으로 제시하여 엘리먼트를 찾기
  • jQuery 정의 필터 셀렉터를 이용하여 폼 요소 다루기-:button, :text, :password
  • 여러 개의 취미를 체크박스에서 선택하기-:checkbox
